# 用户购买意向预测分析报告

## 项目简介

基于用户历史行为数据（浏览时长、页面数、会员状态、加购次数等），构建机器学习模型预测用户是否会购买商品。本报告包含：

- 数据探索与特征工程
- 逻辑回归与随机森林模型对比
- ROC曲线、混淆矩阵、特征重要性分析
- **交互式可视化报告**（手机/PC均可查看）

## 在线报告

- `build_dashboard.py` - 数据生成、建模、报告生成脚本
